📐 공간 단위 처리 안내
• 건축물·용도지역: PROM 폴리곤, 건축물은 대표점(representative_point) 기준으로 구역 포함 여부 결정 / 용도지역은 필지 단위(국토계획법), 건축물 용도는 건축물 단위(건축법)
• 인구·가구·사업체(구역 내): SGIS 집계구 경계(2025년 2분기) × 구역 폴리곤 공간필터 → 구역 내 집계구 합산 (판교 3개·청라 67개)
• 등시간권: 지하철 네트워크 전용(버스·도보 제외). 이소크론 폴리곤(역 500m 버퍼 합집합) × 집계구 대표점 공간조인 → 해당 집계구 총인구 합산 (SGIS 2024)
• 용도지역 색상: 국토부 도시계획 색상 기준 참고하여 설정 (공식 기준과 일부 상이할 수 있음)
